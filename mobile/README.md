# Cognitive Kids - Mobile App

Мобильное приложение для развития когнитивных способностей детей дошкольного возраста (2-6 лет).

## Технологии

- React Native (Expo)
- React Navigation
- Axios для API запросов
- AsyncStorage для локального хранения
- Expo Linear Gradient

## Установка

1. Установите зависимости:
```bash
cd mobile
npm install
```

2. Настройте API URL в `src/config/api.js`:
- Для Android эмулятора: `http://10.0.2.2:5000/api`
- Для iOS симулятора: `http://localhost:5000/api`
- Для физического устройства: замените на IP вашего компьютера в локальной сети

## Запуск

### Web
```bash
npm run web
```

### Android
```bash
npm run android
```

### iOS
```bash
npm run ios
```

Или просто:
```bash
npm start
```

И выберите платформу в Expo Dev Tools.

## Роли пользователей

### 1. Администратор
- Панель управления с общей статистикой
- Управление пользователями
- Создание аккаунтов родителей
- Просмотр активности в приложении

### 2. Родитель
- Создание аккаунтов детей
- Просмотр статистики своих детей
- Получение рекомендаций по развитию
- Мониторинг прогресса

### 3. Ребенок
- 7 развивающих игр
- Система друзей
- Таблица лидеров
- Отслеживание прогресса

## Игры

### 1. Мемори (Memory/Pairs)
**Развивает:** Зрительную память, концентрацию
- Найти парные карточки
- Адаптация по возрасту (от 4 до 8 пар)
- Подсчет ходов и времени

### 2. Найди лишнее (Odd One Out)
**Развивает:** Логическое мышление, классификацию
- Найти предмет, который не подходит
- Сортировка по цвету, форме, категории
- Прогрессивная сложность

### 3. Сортировка (Sorting)
**Развивает:** Классификацию, логику
- Разложить предметы по категориям
- Разные темы (фрукты/овощи, животные, транспорт)
- Интерактивное перетаскивание

### 4. Счет (Counting)
**Развивает:** Математические навыки
- Посчитать предметы
- Выбрать правильное число
- Адаптация к возрасту (до 5, 10 или 15)

### 5. Тени и Силуэты (Shadow Matching)
**Развивает:** Зрительное восприятие
- Найти правильную тень предмета
- Распознавание форм
- Внимание к деталям

### 6. Построй по Образцу (Building)
**Развивает:** Память, пространственное мышление
- Запомнить образец
- Воспроизвести по памяти
- Работа с цветными блоками

### 7. Что будет дальше? (Predicting)
**Развивает:** Причинно-следственное мышление
- Предсказать последствия
- Логические сценарии
- Житейские ситуации

## Особенности

### Адаптация по возрасту
- **2-3 года:** Упрощенные варианты игр
- **4-5 лет:** Средняя сложность
- **6 лет:** Максимальная сложность

### Система мотивации
- Очки за правильные ответы
- Таблица лидеров среди друзей
- Отслеживание прогресса

### Статистика и рекомендации
- Детальная статистика по каждой игре
- Персональные рекомендации для родителей
- Анализ сильных и слабых сторон

## API Endpoints используемые в приложении

### Аутентификация
- `POST /api/auth/login` - Вход
- `GET /api/auth/me` - Текущий пользователь
- `POST /api/auth/register-parent` - Создать родителя (админ)
- `POST /api/auth/register-child` - Создать ребенка (родитель)

### Игры
- `POST /api/games/result` - Сохранить результат
- `GET /api/games/results/:childId` - Результаты ребенка
- `GET /api/games/statistics/:childId` - Статистика ребенка
- `GET /api/games/leaderboard` - Таблица лидеров

### Друзья
- `POST /api/friends/request` - Отправить запрос
- `GET /api/friends/requests` - Получить запросы
- `PUT /api/friends/accept/:requestId` - Принять запрос
- `PUT /api/friends/reject/:requestId` - Отклонить запрос
- `GET /api/friends` - Список друзей

### Администрирование
- `GET /api/admin/users` - Все пользователи
- `GET /api/admin/statistics` - Статистика приложения
- `PUT /api/admin/users/:userId/activate` - Активировать
- `PUT /api/admin/users/:userId/deactivate` - Деактивировать

## Тестовые данные

```bash
cd backend
node scripts/createAdmin.js
```

**Учетные данные администратора:**
- Username: `admin`
- Password: `admin123`

## Разработка

При разработке убедитесь, что:
1. Бэкенд сервер запущен (`npm run dev` в папке backend)
2. MongoDB запущена
3. Правильно настроен API_URL в `src/config/api.js`
