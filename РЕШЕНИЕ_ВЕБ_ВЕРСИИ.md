# Решение: Веб-версия основного приложения

## Проблема:
Expo export:web может не работать для React Native приложения с нативными компонентами.

## Решение:
Создать упрощенную веб-версию основного приложения или использовать другой подход.

## Варианты:
1. Использовать expo start --web для разработки
2. Создать отдельную веб-версию на React (не React Native)
3. Использовать только AR игры на GitHub Pages
